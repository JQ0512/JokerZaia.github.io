第四章.复合类型

4.1 数组

  与C相同，可以存放同一类型的数据。声明数组应指出以下三点

  • 数据类型
  • 数组名称
  • 数组容量
    像这样：int a[10];这就是一个数组的声明。
    数组之所以被称为复合类型，一方面是因为它有多个元素，另一方面是因为它是使用其他类型来创建的。声明数组时必须指明其类型。

  数组的用途基于这样一个事实：可以单独访问数组元素，来利用下标对每个元素进行处理。和C一样，C++的元素也是从0开始编号的。数组声明可以创建一个比较大的变量，然后利用索引标识和访问每个元素。
  数组的初始化在定义时初始化。初始化时注意以下几个问题：

int cards[4]={3,6,8,10};//ok
int cards[4]={};// all zero
int cards[4]={0};//all zero
int cards[4]={3};//card[0]=3, others=0
int cards[4]={2.2};//wrong, int?float?
hand[4]={4,5,6,7};//not allowed
hand=cards;//wrong

另外，在C++的模板库（STL）里面提供了一种数组的替代品——vector

4.2 字符串

  在C++中字符串有两种模式，一种是C模式，一种是自带的string类模式。
  C模式的字符串可以看作一个字符数组，最后一项必须是’\0’作为休止符。当然了，如果我们是直接双引号一句话，数组会自动加上最后的一项’\0’。声明一个字符串的常见方式如下；

char dog[7]={'a','b','c','d','e','f','g','\0'};
char fish[]="fish";

但是在输入字符串的时候这个鬼cin会以空格，回车和制表自动分隔开。那么怎么办呢？用cin.getline()就可以按照行进行输入。
例如：

#include<iostream>
using namespace std;
int main(){
   const int ArSize = 20;
   char name[ArSize];
   char dessert[ArSize];

   cout<<"enter your name:"<<endl;
   cin.getline(name,ArSize);
   cout<<"enter your favorite dessert:"<<endl;
   cin.getline(dessert,Arsize);
   cout<<"I have some delicious "<<dessert<<"for you, "<<name<<".\n";
   return 0;
}

另一种方式是get(), 使用方法（参数）与getline相类似。不过他不会跳过换行符，所以要是碰上了换行，你就咋地呢，就连用两次get()给他pass过去。
比如：cin.get(–).get()，或者cin.get(–);cin.get();
  另一字符串声明方式是使用string类。使用string对象的方式与字符串数组相同。

  • 可以使用C字符串初始化string对象
  • 使用cin将键盘输入存到string 对象中
  • cout显示string 对象
  • 数组表示法同样可以访问string对象里面的字符串

相比于C风格的数组型字符串，C++的类版本字符串更加的方便。首先，不同于数组，由于string是个对象，它是可以赋值的。另外，string可以复制，拼接等等操作，非常方便。拼接用个加号就可以了。

#include<iostream>
#include<string>
#include<cstring>
using namespace std;
int main(){
   char charr1[20];
   char charr2[20]="jaguar";
   string str1;
   string str2="panther";

   //assignment for string objects and character arrays
   str1=str2;//c++ copy
   strcpy(charr1,charr2);//c copy
   
   //appending for string objects and character arrays
   str1+="paste";//c++ add
   strcat(charr1,charr2);//c add
   
   //find the length of a string
   int len1=str1.size();
   int len2=strlen(charr1);

   cout<<str1<<len1<<endl;
   cout<<charr1<<len2<<endl;
   return 0;
}

4.3 结构

C++的结构体与C类似。也是那样一种定义方式

struct inflatable{
   std::string name;
   float volume;
   double price;
}

然后我们初始化一个它就可以有这样几种方式：

1.只声明不赋值
struct{
  ******
}my_smith;
2.声明后赋值
struct{
  ******
}my_smith={
  ******
};
3.后面利用属性赋值
struct{
   int num;
   char t[5];
}x;
x.num=1;

与多维数组相类似，也有结构体数组，毕竟我们已经把各种不同类型的数据集合到了一个结构体里面。而且也会有一些特殊的例子例如位字段。

最后它介绍了文本文件的操作（这是要把C基础全复习一遍的节奏哇）
文件输出条件有这些：

  • 必须有头文件fstream，它定义了一个用于处理输出的ofstream类
  • 需要自己声明对象，自己命名
  • 必须要std
  • ofstream对象与文件能够关联。方法之一就是使用open方法
  • 使用完毕记得用close()关闭输入输出流
  • 可以结合<<输出数据

步骤如下：

  1. 包含头文件fstream
  2. 创建一个ofstream对象
  3. ofstream与对象关联
  4. 像cout一样去使用

而文件的输入满足以下条件：

  • 必须有头文件fstream，它定义了一个用于处理输入的ifstream类
  • 需要自己声明对象，自己命名
  • 必须要std
  • ifstream对象与文件能够关联。方法之一就是使用open方法
  • 使用完毕记得用close()关闭输入输出流
  • 可以结合>>输入数据
  • 读一个用get()，一行用getline()
  • eof(), fail(), 用于判断输入是否成功
//输出
#include <iostream>
#include <fstream>
using namespace std;

int main(){
    //标准输出（写到显示屏幕上）
    char ch[] = "hello world!";
    cout << ch << endl; 
    //文件输入（写到文件里）
    ofstream fout;
    fout.open("data.txt");//将fout对象和文件绑定起来（）
    fout << ch << endl;
    fout.close();
    return 0;
}

//输入
#include <iostream>
#include <fstream>
using namespace std;

int main(){
    //标准输入
    char ch;
    cin >> ch;
    cout << ch << endl;
    //文件输入
    ifstream fin("data.txt");//可以直接在声明中创建对象
    if(!fin.is_open()){
        printf("open error!\n");
        return -1;
    }
    while(fin.good()){
        fin >> ch;
        cout << ch ;
    }

    if(fin.eof()){
        printf("已经读到文件结尾了！\n");
    }else if(fin.fail()){
        printf("判断最后一次读取数据的时候是否遇到了类型不配的情况");
    }else{//fin.bad()
        printf("出现意外的问题，如文件受损或硬件故障，最后一次读取数据的时候发生了这样的问题\n");
    }
    fin.close();
    return 0;
}
